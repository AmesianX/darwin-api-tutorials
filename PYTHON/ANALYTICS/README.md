## DARWIN Data Analytics for Algorithmic Trading

### Prerequisites
1. FTP access to the raw DARWIN data repository. To request it, please visit: https://www.darwinex.com/data/darwin-data
1. Python **v3.6+**

### Python module requirements
1. os
1. io
1. pandas
1. ftplib
1. matplotlib
